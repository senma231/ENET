# FRP GUI客户端使用说明

## 简介
FRP GUI是一个基于.NET开发的FRP客户端图形界面，用于简化FRP的配置和使用过程。

## 系统要求
- Windows 7/8/10/11
- 不需要额外安装.NET运行时（已内置）

## 使用方法

### 基本配置
1. 服务器地址：填写FRP服务器的IP地址或域名
2. 服务器端口：填写FRP服务器的端口号
3. 认证令牌：填写连接FRP服务器所需的认证令牌
4. frpc路径：选择frpc.exe的路径（如果没有，请先下载frpc.exe）

### 代理配置
1. 点击"添加代理"按钮添加新的代理配置
2. 填写代理名称、本地IP、本地端口、远程端口和自定义域名（可选）
3. 添加完成后，代理会显示在列表中

### 操作
- 启动：启动FRP客户端
- 停止：停止FRP客户端
- 保存：保存当前配置
- 导入：导入已有的配置文件
- 导出：导出当前配置为文件

### 高级功能
- 开机自启动：勾选后，应用程序将在系统启动时自动运行
- 注册为系统服务：勾选后，应用程序将作为系统服务运行

## 日志
应用程序底部的日志区域会显示FRP客户端的运行日志，方便排查问题。

## 注意事项
1. 首次使用时，需要配置frpc.exe的路径
2. 如需使用系统服务功能，请确保以管理员身份运行应用程序
3. 日志文件保存在：C:\Users\[用户名]\AppData\Roaming\FrpcGui\Logs

## 常见问题
1. 无法连接服务器：检查服务器地址、端口和认证令牌是否正确
2. 代理无法使用：检查本地端口是否被占用，远程端口是否配置正确
3. 启动失败：检查frpc.exe路径是否正确，尝试以管理员身份运行

## 下载frpc
如果您还没有frpc.exe，可以从以下地址下载：
https://github.com/fatedier/frp/releases

选择适合您系统的版本，解压后获取frpc.exe文件。
