# ENET

ENET是一个基于.NET开发的FRP图形客户端，用于管理和配置FRP代理。

## 功能特点

- 简洁直观的图形界面
- 支持多种代理类型（TCP、UDP、HTTP、HTTPS、STCP）
- 支持多配置文件管理
- 支持多窗口同时运行不同配置
- 系统托盘最小化
- 开机自启动
- 注册为系统服务
- 详细的日志记录

## 系统要求

- Windows 7/8/10/11
- .NET 8.0 或更高版本

## 安装方式

### 安装版

1. 下载最新的安装包 `ENET_Setup.exe`
2. 运行安装程序，按照向导完成安装
3. 从开始菜单或桌面快捷方式启动应用程序

### 便携版

1. 下载最新的便携版 `ENET_Portable.zip`
2. 解压到任意位置
3. 运行 `ENET.exe` 启动应用程序

## 使用说明

1. 基本配置：设置服务器地址、端口和认证令牌
2. 添加代理：点击"添加代理"按钮，配置代理参数
3. 启动服务：点击"启动"按钮开始连接服务器

## 注意事项

某些杀毒软件可能会误报frpc.exe为恶意程序，这是误报。如果安装或运行过程中遇到杀毒软件拦截，请将程序添加到杀毒软件的白名单中。
